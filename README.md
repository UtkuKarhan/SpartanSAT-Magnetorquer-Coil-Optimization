# SpartanSAT-Magnetorquer-Coil-Optimization
Master's Project at San Jose State. Designed a magnetorquer based control system for a 2U satellite effort by SJSU students called SpartanSAT.
This project presents the design, modeling, simulation, and experimental validation of 
optimized magnetorquer coils for the SpartanSAT 2U CubeSat, with the goal of providing reliable 
attitude control in Low Earth Orbit. Traditional control systems struggle to obtain accurate data of 
the crescent moon due to severe illumination gradients. Thus, this work investigates a 
magnetorquer based Attitude Determination Control System as an alternative approach for 
spacecraft orientation and cislunar navigation support. The study develops multiple coil 
configurations and evaluates their performance using MATLAB orbital simulations, and 
International Geomagnetic Reference Field geomagnetic characterization. Control algorithms, 
including B-dot detumbling logic, are designed and optimized to balance magnetic dipole output, 
power consumption, and current limits. Coils in simulation undergo verification and calibration to 
validate analytical and simulation results. The findings provide a complete design workflow for 
magnetorquer systems on small satellites and demonstrate an optimized magnetic actuation 
approach capable of supporting the mission’s unique lunar-observation requirements. 

There are multiple MATLAB .m code files attached to this repo. The main file is named "main". It is important to download and save all the .m files in the same location and run the "main" file for an optimized simulation.
